📝 Widget de Testimonios

Ejemplo de implementación del widget de testimonios en un sitio externo

1️⃣ Pasos de Implementación

Paso 1: Agregar referencias

Copia este código en tu HTML donde quieras mostrar los testimonios:

<div id="testimonials-widget"></div> <script src="http://localhost:3001/widget/testimonials.js"></script>

Paso 2: Inicializar el widget

Luego, inicializa el widget con tu configuración:

<script> TestimonialsWidget.init({ containerId: 'testimonials-widget', apiUrl: 'http://localhost:3000', // URL de tu backend widgetUrl: 'http://localhost:3001', // URL del servicio widget theme: 'light', // 'light' o 'dark' limit: '6', // Límite de testimonios (defecto) featured: false // Mostrar solo destacados }); </script>

Paso 3: ¡Listo!

El widget se mostrará automáticamente en el contenedor con ID #testimonials-widget

⚙️ Opciones de Configuración

Opción Tipo Descripción
containerId string ID del elemento HTML donde se incrustará el widget (REQUERIDO)
apiUrl string URL de tu API backend (ej: https://api.tudominio.com)
widgetUrl string URL del servicio widget (ej: https://widgets.tudominio.com)
theme string 'light' (defecto) o 'dark' - color del tema
limit string Número de testimonios a mostrar por defecto (máx 50)
featured boolean Si es true, muestra solo testimonios destacados

📊 Funcionalidades Incluidas

  • ✅ Testimonios ordenados por destacados (isFeatured)
  • ✅ Muestra el contenido completo del testimonio
  • ✅ Registra automáticamente cada click en un testimonio
  • ✅ Registra automáticamente las vistas
  • ✅ Soporte para múltiples tipos: texto, imagen, video (YouTube)
  • ✅ Tema claro/oscuro personalizable
  • ✅ Botón "Ver todos" para mostrar más testimonios
  • ✅ Auto-ajuste de altura del iframe
  • ✅ Responsive - se adapta a cualquier tamaño de pantalla

💡 Ejemplos Adicionales

Tema Oscuro:

TestimonialsWidget.init({ containerId: 'my-testimonials', apiUrl: 'https://api.midominio.com', widgetUrl: 'https://widgets.midominio.com', theme: 'dark' });

Con límite personalizado:

TestimonialsWidget.init({ containerId: 'testimonials', apiUrl: 'https://api.midominio.com', widgetUrl: 'https://widgets.midominio.com', limit: '12' // Mostrar 12 testimonios por defecto });

Solo testimonios destacados:

TestimonialsWidget.init({ containerId: 'featured-testimonials', apiUrl: 'https://api.midominio.com', widgetUrl: 'https://widgets.midominio.com', featured: true });

✨ ¡Widget Listo!

El widget está completamente funcional. Los testimonios se cargan automáticamente desde tu API, registra vistas y clicks, y se ajusta responsivamente a cualquier tamaño de pantalla.
